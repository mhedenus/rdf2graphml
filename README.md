# rdf2graphml

Python library for easy converting any __RDF__ to __GraphML__ (http://graphml.graphdrawing.org/specification/xsd.html)
compatible with __yEd__ (https://www.yworks.com/products/yed).

Following styles are known by __yEd__:

`shape`:
- `diamond`
- `ellipse`
- `hexagon`
- `octagon`
- `parallelogram`
- `rectangle`
- `rectangle3d`
- `roundrectangle`
- `trapezoid`
- `trapezoid2`
- `triangle`

`line_type`:
- `dashed`
- `dashed_dotted`
- `dotted`
- `line`

`target_arrow`:
- `diamond`
- `none`
- `standard`
- `transparent_circle`
- `white_delta`
